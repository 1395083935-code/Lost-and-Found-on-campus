# 校园失物招领系统 v1.0.0 - 上线通知

---

## 📢 重要公告

亲爱的校园用户们：

**校园失物招领系统** (Campus Lost & Found) 即将于 **2026 年 5 月 7 日 (星期三)** 正式上线！🎉

该系统旨在帮助校园师生快速找到遗失的物品，建立一个高效的失物招领平台。

---

## 🎯 功能一览

### 1️⃣ **物品浏览与查询**
- 📱 浏览已发布的失物/招领信息
- 🔍 按物品类别 (证件、电子产品、衣物、书籍等) 筛选
- 📍 看到清晰的物品图片和详细描述
- ⏱️ 查看物品的发布时间和地点

### 2️⃣ **智能搜索**
- 💡 搜索热词建议 (如 "学生证"、"AirPods"、"iPhone 14")
- 🔎 按关键词快速搜索相关物品
- 📊 查看热门搜索关键词，了解校园失物趋势

### 3️⃣ **我的物品**
- ➕ 发布我找到的物品
- ❤️ 收藏感兴趣的物品
- 👤 管理我的个人信息
- 📋 查看我发布过的所有物品

### 4️⃣ **智能推荐**
- 🤖 根据您的收藏历史推荐相关物品
- 📈 新发布的物品会优先展示
- 💬 与失物主人直接联系

## 📱 如何使用？

### 步骤 1: 访问系统
- 电脑: https://campus.edu.cn/lostfound
- 手机: 在浏览器中访问上述网址 (完全适配移动设备)
- 或扫描二维码: [二维码待生成]

### 步骤 2: 登录账户
- **推荐**: 使用校园微信一键登录 (最便捷)
- **备选**: 使用学号 + 教职工号 + 密码登录
- **首次用户**: 点击"新用户注册"

### 步骤 3: 浏览物品
1. 进入首页，看到失物招领的瀑布流列表
2. 点击任意物品卡片，查看完整信息
3. 如果是您要找的，直接点击"联系失物主人"

### 步骤 4: 发布物品
1. 点击"我的" → "发布物品"
2. 选择类别 (失物 or 招领)
3. 上传清晰的物品照片
4. 填写详细描述 (颜色、特征、发现地点等)
5. 点击"发布"

### 步骤 5: 搜索物品
1. 点击首页搜索框
2. 输入关键词 (如 "iPhone")
3. 系统会自动显示相关物品
4. 如果没找到，查看热词建议

## 🎁 功能示例

### 场景 1: 找到一串钥匙
```
1. 打开系统，点击"发布物品"
2. 选择"招领" → 类别"钥匙"
3. 上传钥匙的清晰照片
4. 填写："在图书馆 3 楼自习室找到，带有蓝色钥匙套，上有红色塑料标签"
5. 点击"发布"
6. 系统自动通知失物主人，失物主人看到您的发布后会与您联系
```

### 场景 2: 找不到学生证
```
1. 打开系统，点击"搜索"
2. 输入"学生证"，系统显示最近 7 天发布的学生证
3. 看到一张照片相符的学生证，点击进入详情
4. 点击"联系失物主人"，填写您的信息
5. 失物主人收到通知，通过系统与您沟通
6. 约定地点取回物品
```

### 场景 3: 我想收藏某件物品
```
1. 浏览物品列表或搜索结果
2. 看到感兴趣的物品，点击"❤️ 收藏"
3. 物品自动添加到"我的收藏"
4. 如果有新的类似物品发布，系统会推荐给您
5. 随时可以从"我的"中查看已收藏的物品
```

## ❓ 常见问题 (FAQ)

### 账户与登录
**Q: 如何登录？**
A: 支持 3 种方式：
   1. 微信一键登录 (推荐，最快)
   2. 学号 + 密码登录 (本科生/研究生)
   3. 教职工号 + 密码登录 (教职工)

**Q: 忘记密码怎么办？**
A: 点击登录页的"忘记密码"，按提示操作即可。将向校内邮箱发送重置链接。

**Q: 账户安全吗？**
A: 完全安全。密码采用加密存储，支持 HTTPS 全程加密通信。您的隐私信息受校园数据保护政策保护。

### 功能使用
**Q: 可以发布的物品有限制吗？**
A: 无限制。但需要遵守校园规定：不能发布违禁品、危险品、盗窃品等。

**Q: 发布后可以修改吗？**
A: 可以。进入"我的物品"，点击编辑图标即可修改。

**Q: 物品多久会被删除？**
A: 默认保留 30 天。超过 30 天的物品会自动归档。您也可以手动删除。

**Q: 搜索关键词会被记录吗？**
A: 会的。我们用于优化搜索推荐，但不会分享给第三方。

### 联系与隐私
**Q: 如何与失物主人联系？**
A: 通过系统内置的"联系失物主人"按钮。系统会显示他们的校内身份 (学号/教职工号的部分字符)，您可以在系统内聊天。

**Q: 我的个人信息会被泄露吗？**
A: 不会。失物主人只能看到您的校内身份和聊天消息，看不到您的真实姓名、电话等隐私信息。

**Q: 可以删除我发布的物品吗？**
A: 可以。进入"我的物品"，点击物品卡片右上的"删除"按钮。

### 技术问题
**Q: 系统卡顿或显示异常怎么办？**
A: 
   1. 尝试刷新页面 (F5 或 Ctrl+R)
   2. 清空浏览器缓存 (Ctrl+Shift+Delete)
   3. 切换到其他浏览器测试
   4. 检查网络连接（断开 WiFi 后重新连接）
   5. 如果问题仍未解决，请联系技术支持

**Q: 手机端和电脑端的数据是否同步？**
A: 完全同步。同一账户在任何设备上的操作都会实时反映。

**Q: 系统支持离线使用吗？**
A: 暂不支持。需要网络连接才能使用。
