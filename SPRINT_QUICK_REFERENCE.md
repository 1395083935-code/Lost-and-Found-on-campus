## 🚨 遇到问题快速查询

### 问题1：集成测试失败
```
症状: 请求404或500错误
排查步骤:
1. 检查后端是否启动: ps aux | grep java
2. 检查数据库是否连接: mysql -uroot -p123456 -e "SHOW DATABASES;"
3. 查看后端日志: tail -f logs/campus-backend.log
4. 检查API是否在白名单中: 查看JwtInterceptor
解决方案: 参考 SPRINT_DETAILED_EXECUTION_PLAN.md 任务1
```

### 问题2：数据库查询缓慢
```
症状: 响应时间>1000ms
排查步骤:
1. 使用EXPLAIN分析查询: EXPLAIN SELECT ...
2. 检查索引是否存在: SHOW INDEX FROM table_name
3. 查看slow query log: tail /var/log/mysql/slow.log
解决方案: 参考 SPRINT_DETAILED_EXECUTION_PLAN.md 任务2/4
```

### 问题3：中文显示乱码
```
症状: 显示 "ææ‰‹æœº" 代替 "手机"
排查步骤:
1. 检查数据库字符集: SELECT @@character_set_database;
2. 检查表字符集: SHOW CREATE TABLE table_name;
3. 检查application.yml: server.servlet.encoding.charset=UTF-8
解决方案: 参考 SPRINT_DETAILED_EXECUTION_PLAN.md 任务3
```

### 问题4：权限拒绝错误
```
症状: 非管理员访问管理功能返回"权限不足"
排查步骤:
1. 检查用户role是否是管理员
2. 检查JwtInterceptor是否正确验证
3. 查看Controller是否有权限检查代码
解决方案: 参考 SPRINT_DETAILED_EXECUTION_PLAN.md 任务3
```
