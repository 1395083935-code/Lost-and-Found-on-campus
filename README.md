# Lost-and-Found-on-campus

基于校园失物招领的全栈开发工作区，包含：

- `campus-backend`：Spring Boot 后端
- `campus-frontend`：Vue 前端
- 根目录下部署与运维文档

## Quick Start

### Backend

```bash
cd campus-backend
mvn spring-boot:run
```

### Frontend

```bash
cd campus-frontend
npm install
npm run serve
```

## Notes

- Backend default port: `8082`
- Frontend default port: `8081`
