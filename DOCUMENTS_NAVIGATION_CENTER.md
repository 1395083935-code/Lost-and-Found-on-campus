## 🚨 问题快速排查指南

### 遇到问题时的处理流程

```
第一步: 确认问题类型
  → 技术问题 (代码/工具) ✓
  → 资源问题 (硬件/网络) ✓
  → 流程问题 (沟通/权限) ✓

第二步: 查询快速参考卡
  位置: SPRINT_QUICK_REFERENCE.md - "🚨 遇到问题快速查询"

第三步: 按照解决方案排查
  - 查看症状描述是否匹配
  - 执行排查步骤
  - 尝试提供的解决方案

第四步: 记录在缺陷数据库
  - 问题描述
  - 重现步骤
  - 尝试过的解决方案
  - 当前状态 (已解决/待解决)

第五步: 如果无法自解，升级
  - 通知项目经理
  - 提供缺陷数据库链接
  - 不要继续花时间调试
```

### 常见问题速查表

| 问题 | 快速解决 | 详细文档 |
|------|---------|---------|
| API返回401/403 | 检查JwtInterceptor白名单 | SPRINT_QUICK_REFERENCE.md |
| 数据库查询慢 | 检查是否有EXPLAIN计划 | SPRINT_DETAILED_EXECUTION_PLAN.md - 任务4 |
| 中文显示乱码 | 检查charset=utf8mb4 | SPRINT_QUICK_REFERENCE.md |
| 编译失败 | 运行mvn clean compile | SPRINT_DETAILED_EXECUTION_PLAN.md |
| 后端启动失败 | 检查数据库连接 | SPRINT_QUICK_REFERENCE.md |
